スライムの状態


気温や配送を考慮して固めに制作しております。

そのためお届け時に固く伸びが不十分に感じる場合がございます。

数日で通常のいい状態になりますので保湿乳液などのクリームは混ぜないでください。

逆に手につく場合は付属のホウ砂水を少量ずつ足して調整してください。


また全ての商品をご注文後に制作しておりますので気泡が残っている場合がございます。

こちらも数日で気泡が抜けいい状態になります。

⚠︎ただしクリアスライムの場合は1〜2週間かかる場合がございます。


スライムを触る前に


スライムを触る際は必ず手を綺麗に洗ってください。

不衛生な状態で触ると腐りやすくなったり手にスライムがつきやすくなります。


スライムの触り方


指でタップしたり伸ばして空気を含ませバブル音を鳴らしたりと自由に楽しんでください♡♡


スライムの保管方法


ハンドメイドスライムは特に気候により状態が変化します。

直射日光が当たらない涼しい場所で保管してください。

水滴が発生したりトロトロに溶けたりし腐る原因になりますのでご注意ください。

夏場は冷蔵庫にて保管をおすすめします。


その他の注意


スライムの対象年齢は6歳以上と言われております。

小さなお子様やペットがいるご家庭は誤飲にお気をつけください。

万が一事故が起きた場合も当店では責任を負いかねますのでご了承ください。

PVA / ボンド / ホウ砂 / 樹脂粘土 / 紙ねんど / 保湿ローション / 人工雪 / 香料にアレルギーのある方はご使用をお控えください。

肌に合わない場合は直ちに使用を中止してください。
